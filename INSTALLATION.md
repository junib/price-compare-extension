# Quick Installation Guide

## Step 1: Create Icons (Optional)
Create 3 PNG files in the `icons/` folder:
- `icon16.png` (16x16)
- `icon48.png` (48x48)  
- `icon128.png` (128x128)

You can skip this step - the extension will work with default Chrome icons.

## Step 2: Load Extension in Chrome

1. Open Chrome
2. Go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top-right corner)
4. Click **"Load unpacked"**
5. Select the `price-compare-extension` folder
6. Done! âœ…

## Troubleshooting

- **Extension not loading?** Make sure you selected the correct folder (price-compare-extension)
- **Button not showing?** Refresh the page after installing
- **Panel not appearing?** Check browser console (F12) for errors
