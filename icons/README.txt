ICON FILES NEEDED
=================

Please create the following PNG icon files in this directory:

1. icon16.png  (16x16 pixels)
2. icon48.png  (48x48 pixels)
3. icon128.png (128x128 pixels)

You can:
- Use an online icon generator (like favicon-generator.org)
- Create simple colored squares with a shopping/food theme
- Use any image editor

The extension will work without these icons, but Chrome will show a default icon.

Suggested design: Shopping cart + food emoji or purple gradient background
